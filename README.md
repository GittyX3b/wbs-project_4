# WBS-Project 4: Travel Agency

## React-App using react-router

### Objective

Set up a basic skeleton for a Travel Agency website using React, React Router, Vite, TailwindCSS, and DaisyUI. This exercise focuses on structuring a modern front-end project and practising fundamental navigation and UI layout skills using React Router.

### Instructions

Follow the functional requirements below to complete the exercise:

Set up a new React app using Vite, with TailwindCSS and DaisyUI. Refer to our Our Basic Setup Guide in the course materials for step-by-step instructions.
Install and configure React Router in your project.

Use DaisyUI components to quickly scaffold the UI.

The website should include the following pages:

Home

Destinations

Destinations/:slug

About (Information about the company)

Contact (Simple contact form (no submission logic required))

On the Home page:

Display a search bar with criteria such as origin, destination, dates, etc.

When the Search button is clicked, navigate to the Destinations page.

On the Destinations page:

Hard-code a list of travel destination cards (e.g. country or city previews), use slugs as identifiers.
